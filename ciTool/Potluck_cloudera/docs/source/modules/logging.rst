Logging
*******

.. toctree::
   :maxdepth: 2

Potluck has extensive support for embedding logging information in the test scripts.
By default, all the commands executed on a device and their outputs will be logged.

Don't shy away from putting log messages in your test scripts. Logging helps while analyzing the logs of a failed testcase.
